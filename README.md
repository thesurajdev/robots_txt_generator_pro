# Robots.txt Generator Pro

A professional-grade, high-performance robots.txt generator designed for SEO specialists and web developers. This tool provides precise control over crawler behavior with a minimalist, high-contrast interface.

## 🚀 Features

- **Multi-Agent Management**: Configure rules for multiple user-agents (Googlebot, Bingbot, etc.) in one place.
- **Live Preview**: Real-time generation of your `robots.txt` file as you edit.
- **ASCII Art Integration**: Add a professional touch to your files with a built-in ASCII art generator.
- **Sitemap Support**: Easily add and manage multiple sitemap URLs.
- **Mobile Responsive**: Fully optimized for mobile devices with a custom hamburger menu and floating navigation.
- **One-Click Export**: Copy to clipboard or download the `.txt` file instantly.

## 🛠️ Tech Stack

- **Framework**: Next.js 15+ (App Router)
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion
- **Icons**: Lucide React

## 📖 Getting Started

1. Launch the [Generator Tool](/tool).
2. Configure your Global settings (Header/Footer).
3. Add User Agents and define their Allow/Disallow rules.
4. Add your Sitemap URLs.
5. Copy or Download your optimized `robots.txt` file.
